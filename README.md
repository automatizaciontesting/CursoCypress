# Ambientación maquina para Cypress en windows 10

Esta actividad de ambientación de maquina permite que pueda utilizar el codigo que se encuentra en este proyecto y poder realizar las ejecuciones de forma adecuada. Esta configuración permite realizar los minimos pasos que son crear, ejecutar y publicar reportes para **Cypress**.
